# Stock Market Analysis – JK Cement

## Overview
Analyzed historical stock price and volume data for JK Cement to evaluate market behavior, investment risk, and key performance indicators using financial and statistical techniques.

## Key Features
- 576 Days of Historical Data Analysis
- Beta Calculation vs Nifty 50
- Regression Analysis
- Risk Modelling using Historical Simulation
- Price & Volume Trend Analysis
- Investment Risk Assessment

## Skills Demonstrated
- Financial Analysis
- Equity Research
- Risk Modelling
- Statistical Analysis
- Python
- Excel

## Results
- Beta: 0.25 (Defensive Stock Characteristics)
- Regression R²: 0.031
- Historical Simulation Price Range: ₹2,583 – ₹5,262
